Inicio HTML CSS

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje que usamos para crear páginas web. Es como las instrucciones que le damos a un navegador para mostrar el contenido (texto, imágenes, botones, etc.) de una página. Se podría decir que es el "esqueleto" de una página

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje que se usa para darle estilo a las páginas web. Si HTML es el que se encarga de la estructura y el contenido (como el texto, imágenes, y enlaces), CSS es el que se encarga de que ese contenido se vea bien, es decir, que la página sea visualmente atractiva.
