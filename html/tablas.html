Tablas

Una tabla es una estructura que se usa para mostrar datos en filas y columnas, de manera organizada y fácil de leer. Las tablas son útiles cuando necesitas mostrar información estructurada.

La estructura básica de una tabla es la siguiente:

  • "table": Esta etiqueta indica que estás creando una tabla.
  • "tr" (table row): Define una fila en la tabla.
  • "td" (table data): Define una celda en la tabla, que contiene datos.
