Elementos

Los elementos son componentes discretos dentro de un documento HTML que generalmente constan de al menos una etiqueta, pero pueden tener dos etiquetas con contenido opcional. El nombre de la etiqueta determina cómo se representará el elemento en el documento HTML.

  • <p>: Es el atributo más importante, ya que indica la fuente o la ubicación de la imagen. Puede ser una URL o un archivo local.
  • Este es un párrafo: Describe el contenido de la imagen
  • </p>: Describe el contenido de la imagen
