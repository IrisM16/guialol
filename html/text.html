Text Formatting

Podemos cambiar la apariencia del texto dentro de una página web. Con etiquetas especiales, puedes hacer que el texto se vea negrita, cursiva o subrayado.

Los siguientes elementos HTML pueden afectar el formato del texto:

  • <b>: Sirve para resaltar una palabra o frase haciéndola más gruesa o en negrita.
  • <em>: Sirve para poner el texto en cursiva (letras inclinadas)
  • <u>: Sirve para subrayar el texto
  • <s>: Sirve para tachar el texto.
