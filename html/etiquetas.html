Etiquetas

En HTML, todo se organiza usando etiquetas. Las etiquetas son como instrucciones que le dicen al navegador cómo debe mostrar el contenido.

Cada etiqueta tiene un propósito diferente: algunas sirven para mostrar un título, otras para mostrar párrafos, imágenes, enlaces, o listas. Las etiquetas de HTML siempre van entre los símbolos < >.

Algunas etiquetas comunes:

  • <html>
  • <head>
  • <tittle>
  • <body>
  • <h1> - <h6>
  • <p>
  • <a>
  • <img>
  • <ul>, <ol> y <li>
  • <div>
