Inicio HTML CSS

Los atributos HTML son datos que se pueden agregar a la etiqueta de apertura de un elemento. Estos atributos modifican el comportamiento predeterminado del elemento o brindan información adicional sobre él.
Los atributos se escriben dentro de la etiqueta de apertura y le dan más información a esa etiqueta.
Siempre siguen este formato:

Algunos ejemplos son:

Atributo href (enlaces): Si tienes una etiqueta <a>, que sirve para crear enlaces, puedes usar el atributo href para decir a dónde va ese enlace.

Atributo href (enlaces): Si tienes una etiqueta < a >, que sirve para crear enlaces, puedes usar el atributo href para decir a dónde va ese enlace.
