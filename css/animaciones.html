Animaciones

Las animaciones en CSS permiten agregar movimiento y efectos visuales a los elementos de una página web

@keyframes

Controla una animación con múltiples pasos intermedios, definiendo los valores de las propiedades en cada fotograma clave.

animation

Propiedad abreviada que establece las animaciones de un elemento.

animation-delay

Define cuándo comienza una animación.

animation-direction

Determina si una animación se ejecuta hacia adelante o hacia atrás en algunos o todos sus ciclos.

animation-duration

Define cuánto tiempo debe tardar una animación en completarse.

animation-fill-mode

Define qué valores aplica una animación fuera de su tiempo de ejecución.

animation-iteration-count

Define cuántas veces se ejecuta una animación.

animation-name

Define una lista separada por comas de animaciones para aplicar al selector dado.

animation-play state

Define si una animación está en ejecución o en pausa.

animation-timing-function

Da forma a la curva de velocidad de la animación.
