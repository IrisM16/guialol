Funciones Matemáticas

sirven para realizar cálculos directamente dentro del código CSS, lo que permite diseñar elementos de manera más dinámica y adaptable sin tener que calcular valores exactos manualmente.

  • margin: Propiedad abreviada para margin-top, margin-bottom, margin-left y margin-right para crear espacio alrededor de un elemento o para cada lado individualmente.
  • margin-bottom: Define el área del margen inferior de un elemento.
  • margin-left: Define el área del margen izquierdo de un elemento.
  • margin-right: Define el área del margen derecho de un elemento.
  • margin-top: Define el área del margen superior de un elemento.
