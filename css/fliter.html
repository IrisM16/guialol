Filter

La propiedad filter en CSS aplica efectos gráficos como el desenfoque a un elemento.

  • blur: Define el radio de desenfoque de un elemento.
  • brightness: Define el brillo u oscuridad de un elemento.
  • contrast: Define el contraste de un elemento.
  • drop-shadow: Crea un efecto de sombra proyectada sobre un elemento.
  • grayscale: Convierte un elemento a escala de grises.
  • hue-rotate: Acepta un valor de ángulo y rota el tono de un elemento.
  • invert: Invierte los colores de un elemento.
  • opacity: Controla qué parte del fondo es visible a través del elemento al que se aplica.
  • saturate: Aumenta o disminuye la intensidad del color de un elemento.
  • sepia: Cambia el color de un elemento a sepia.
