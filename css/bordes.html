Bordes

Un borde es una línea que rodea un elemento, como un marco alrededor de una pintura. Estos ayudan a definir el contorno de un elemento en la página web.

Los bordes se pueden configurar con un ancho, estilo y color específicos:

  • width: El grosor del borde. Se puede configurar en píxeles o con una de las siguientes palabras clave: thin, medium, o thick.
  • style: El diseño del borde, algunos de estos estilos incluyen: none, dotted, y solid.
  • color: El color del borde. Se pueden usar nombres de colores, valores hexadecimales, RGB o HSL.

border

Cambia el color de fondo.

border-bottom

Establece una imagen como fondo.

border-color

Controla si la imagen de fondo se repite o no.

border-left

Define la posición de la imagen de fondo.

border-radisus

Controla el tamaño de la imagen de fondo.

background-right

Define si la imagen de fondo se desplaza con el contenido de la página o se queda fija.

background-style

Establece el estilo del borde. El valor predeterminado es ninguno.

background-top

Propiedad abreviada que define el ancho, el color y el estilo del borde superior de un elemento.

background-width

Propiedad abreviada que establece el ancho del borde general de un elemento o de cada lado individualmente.
