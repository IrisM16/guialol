Colores

La propiedad "color" CSS establece el color del texto:

  • color: Para establecer valores de propiedad de color para un elemento.
  • hsl(): Define el color proporcionando el valor de tono, el valor de saturación y el valor de luminancia.
  • hsla(): Define el color proporcionando el valor de tono, el valor de saturación, el valor de luminancia y un valor alfa opcional (transparencia).
  • opacity: Para medir qué tan transparente es un elemento.
  • rgb(): Define un color proporcionando valores para rojo, verde y azul.
  • rgba(): Define el color proporcionando el valor rojo, el valor verde, el valor azul y un valor alfa opcional (transparencia).
