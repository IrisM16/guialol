Funciones Matemáticas

La overflowpropiedad controla lo que sucede con el contenido que se derrama o se desborda fuera de su caja.

Los valores más utilizados son:

  • hidden: Cuando se establece este valor, cualquier contenido que se desborde quedará oculto a la vista.
  • scroll: Cuando se establece este valor, se agregará una barra de desplazamiento al cuadro del elemento para que el resto del contenido se pueda ver desplazándose.
  • visible: Cuando se establece este valor, el contenido desbordado se mostrará fuera del elemento que lo contiene. Tenga en cuenta que este es el valor predeterminado.
