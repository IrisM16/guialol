Grids

Los grids en CSS son un sistema de diseño que te ayuda a organizar elementos en filas y columnas de una manera precisa y adaptable.

  • grid: Forma abreviada para establecer múltiples valores de propiedad de cuadrícula.
  • grid-area: Identifica un elemento dentro de una plantilla de cuadrícula o especificar la ubicación de un elemento dentro de una cuadrícula.
  • grid-auto-columns: Especifica los tamaños de columna predeterminados para un contenedor.
  • grid-auto-rows: Especifica los tamaños de fila predeterminados para un contenedor.
  • grid-column: Especifica la extensión del contenido en un conjunto de columnas dentro de un marco de cuadrícula.
  • grid-column-gap: Especifica el espaciado entre los elementos de columna dentro de un marco de cuadrícula.
  • grid-column-start: Una propiedad que especifica el inicio de un intervalo de contenido sobre un conjunto de columnas dentro de un marco de cuadrícula.
  • grid-row: Especifica la extensión del contenido en un conjunto de filas dentro de un marco de cuadrícula.
  • grid-row-end: Especifica el final de un intervalo de contenido sobre un conjunto de filas dentro de un marco de cuadrícula.
  • grid-row-gap: Una propiedad que especifica el espaciado entre elementos de fila dentro de un marco de cuadrícula.
  • grid-row-start: Una propiedad que especifica el inicio de un intervalo de contenido sobre un conjunto de filas dentro de un marco de cuadrícula.
