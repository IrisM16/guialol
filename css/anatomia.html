Anatomía

En CSS, definimos reglas para decirle al navegador cómo queremos que ciertos elementos de la página se vean. Cada regla CSS tiene una estructura o "anatomía" básica que siempre sigue un formato parecido.

Anatomía de una regla:

Selector: Indica qué elementos de la página queremos modificar.
Declaración: Indica qué cambios (estilos) se aplicarán a esos elementos.

Partes de una declaración

Propiedad: Es lo que quieres cambiar (como el color, el tamaño de la fuente, etc.).
Valor: Es el nuevo valor que le quieres dar a esa propiedad.
