Padding

El espacio entre el contenido de un cuadro y los bordes del mismo se conoce como Padding . El relleno es como el espacio entre una imagen y el marco que la rodea.

  • padding: Propiedad abreviada para padding-top, padding-bottom, padding-left y padding-right que establece el espacio entre el contenido y el margen de un elemento o para cada lado individualmente.
  • padding-bottom: Define el ancho del relleno inferior de un elemento.
  • padding-left: Define el ancho del relleno izquierdo de un elemento.
  • padding-right: Define el ancho del relleno derecho de un elemento.
  • padding-top: Define el ancho del relleno superior de un elemento.
